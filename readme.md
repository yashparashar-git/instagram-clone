# 📸 Instagram Clone (MERN Stack)

A full-stack Instagram Clone built using the MERN stack. This project includes user authentication, profile management, post uploads, likes, stories, search functionality, and more. It is designed to provide a real-world social media experience while strengthening full-stack development skills.

---

## 🚀 Features

- 🔐 User Registration & Login Authentication
- 👤 User Profile
- ✏️ Edit Profile (Name, Bio & Profile Picture)
- 📸 Create & Upload Posts
- ❤️ Like / Unlike Posts
- 👥 Follow & Unfollow Users
- 🔍 Search Users by Username
- 📖 Story Section
- 🖼️ Responsive Feed
- 💾 Save Posts
- 🗄️ MongoDB Database Integration

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/instagram-clone.git
```

### Navigate to Project Folder

```bash
cd instagram-clone
```

### Install Dependencies

```bash
npm install
```

### Start the Server

```bash
npm start
```

or

```bash
nodemon
```
